⚙️ Set up "Send to Google Sheet" (one-time)

1. Open your Hero_Energy_Bookkeeping_Template sheet → Extensions → Apps Script.
2. If you already have the auto-sort script there, just paste this underneath it in the same file. Otherwise paste it in on its own.
3. Click Deploy → New deployment → type: Web app → Execute as: Me → Who has access: Anyone → Deploy. Authorise when prompted.
4. Copy the Web app URL it gives you and paste it into the field below.

Note: because of a browser security quirk with Apps Script Web Apps, this tool can't read back a confirmation — after sending, just glance at the sheet to confirm the rows landed.
